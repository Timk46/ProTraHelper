einfach von klein nach groß gehen, aber umgekehrt nicht. Dann ist es nicht mehr
implizit. Dann muss man es explizit machen mit den runden Klammern. Das heißt, man
kann von Byte auf Short oder von Byte auf Int oder von Byte auf Float gehen, das

sind so die zwei Guidelines, die man nachnehmen könnte, um zu gucken, ob das
