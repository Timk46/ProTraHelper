# CodeGame Java

## Overview
This directory contains the Java implementation of the CodeGame project.

## Setup and Run
Ensure you have Java Development Kit (JDK) installed.

To compile and run the project, run:
```sh
make
```

## Cleanup
Remove the generated files with the following command:
```sh
make clean
```

## Game Grid
The game grid configuration is stored in the `game.grid.txt` file.
