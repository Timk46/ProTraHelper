# CodeGame 

## Setup
Ensure you have a C++11 compatible compiler installed.

`make`

## Run
`./codegame`

## Cleanup
Remove the generated files with the following command:

`make clean`

## Generate Base64 Files

### make it excutiable
`chmod +x convert_to_base64.sh`

### run
`./convert_to_base64.sh ./`
