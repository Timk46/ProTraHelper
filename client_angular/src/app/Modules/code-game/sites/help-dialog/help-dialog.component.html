Allgemeine Informationen

Willkommen im Code-Game! Hier kannst du deine Programmierfähigkeiten testen und verbessern.

So funktioniert's

  1. Schreib deinen Code in den Editor
  2. Klicke auf "Ausführen" um deinen Code zu testen
  3. Beobachte die Animation des Spiels
  4. Überprüfe die Ergebnisse und die Compiler-Ausgabe
