Um Rhino-Dateien direkt aus der Webanwendung zu öffnen, wird die ProTra-Helferanwendung benötigt.

  1. Installation: Falls noch nicht geschehen, laden Sie die Helferanwendung unter diesem Link herunter und installieren Sie sie.
  2. Starten: Stellen Sie sicher, dass die Helferanwendung auf Ihrem Computer gestartet ist.
  3. Rhino-Pfad: Konfigurieren Sie den Pfad zu Ihrer Rhino-Installation im Tray-Menü der Helferanwendung, falls dies noch nicht automatisch geschehen ist.
  4. API-Token: Kopieren Sie das API-Token aus dem Tray-Menü der Helferanwendung und fügen Sie es oben in das vorgesehene Feld ein und speichern es.
