Mithilfe dieses Editors kann ein eigenes Spielfeld erstellt werden. Das Layout des Spielfelds kann durch Klicken auf die Zellen verändert werden. Die Zellen können mit Hindernissen, Gegenständen (zum Aufsammeln), dem Spieler und dem Ziel belegt werden. Zudem können Zellen auf eine Blacklist oder Whitelist gesetzt werden. Abhänigig davon ob eine Zelle nicht besucht werden darf oder besicht werden kann.Ein Objekt kann mehrfach platziert werden, dazu das Objekt im Dropdown auswählen und die Zelle anklicken.
