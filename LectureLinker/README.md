# LectureLinker
- The Lecture Linker provides a Chatbot for each Lecuture linking to specific timestamps from the lecture videos.
- This enables students to ask questions and quickly locate relevant information.

## Fast API Backend
- Hosted on our university server with all the video files so we don't have to store them in this repo.
- You need to be connected to Uni-Siegen VPN.
